# Iron Lady – AI & Technology Internship Assignment

This repository holds my internship assignment submission for the AI & Technology Intern position at Iron Lady.

---

## Task 1: AI-based Customer Interaction Solution

**Project:** Iron Lady AI Career Guide  
**Objective:**  
To help prospective learners understand Iron Lady’s offerings according to their current level (student, fresher, or working professional).

**Features:**
- Basic AI chatbot interface
- Helps users navigate suitable programs
- Enhances user understanding and interaction
- Developed using Python and Flask

---

## Task 2: Internal Business Automation Solution

**Project:** Learner Management System  
**Objective:**  
To support internal teams in managing learner information and avoiding manual tracking.

**Features:**
- Enter learner information (name, email, level, and interest)
- Display all learners
- Delete learner information
- SQLite database management
- Developed using Python, Flask, HTML, and SQLite
